# Game-1

Unityで制作中の六角オセロ型ボードゲームの試作プロジェクトです。

現在は、六角形グリッド上で石を配置し、相手の石を挟んで反転させる基本ルールを実装しています。  
あわせて、合法手の表示、手番管理、パス判定、勝敗判定、リスタート機能まで確認できる状態です。

今後は、この六角オセロをベースに、道路による接続や得点ボーナスを加え、  
国取り・ローグライク要素を持つ独自のボードゲームへ発展させる予定です。

## 開発環境
- Unity
- C#
- Git / GitHub

## 現在の実装内容
- 六角形グリッド盤面の生成
- 石の配置と反転処理
- 合法手判定
- 手番交代 / パス判定
- 勝敗判定
- リスタート機能

- 接続による得点ボーナス
